# 💻 Roteiro 2: Controle de Fluxo em Java

Neste pacote estão os códigos referentes à segunda aula de laboratório. O objetivo aqui foi praticar estruturas condicionais e de repetição com interação do usuário no terminal, incluindo validações de idade e nota, cálculo de média e contagem de alunos aprovados.

## 🛠️ O que foi praticado
* **Classe Scanner**: utilizada para ler entradas de texto e números via terminal.
* **Estruturas condicionais**: uso de if, else if, else e operador ternário para validações e classificações.
* **Estruturas de repetição**: uso de for e while para menus e leitura de dados repetidos.
* **Switch-case**: controle do menu interativo por opções.
* **Validações de entrada**: verificação de idade inválida e notas fora do intervalo de 0 a 10.
* **Cálculo de média**: leitura de 4 notas e exibição da média final.
* **Desafio extra**: leitura de 5 alunos e exibição do total de aprovados.

## ▶️ Como rodar apenas este roteiro
Estando na pasta Roteiro02/src, compile e execute o arquivo principal:
`javac App.java Aluno.java`
`java App`

## 📌 Estrutura principal
* **src/App.java**: Fluxo principal do programa e menu interativo.
* **src/Aluno.java**: Classe com os dados básicos do aluno (nome, idade e nota).
